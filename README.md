# Superstore-Sales-Analysis-Excel
Sales analysis of superstore dataset using excel-Includes data cleaning,KPI creation, pivot tableland interactive dashboard.
THIS PROJECT ANALYZE THE SUPER STORE DATASET TO IDENTIFY SALES TREND AND BUSINESS INSIGHT USING EXCEL.

THE OBJECTIVE IS TO UNDERSTAND

1- SALES BY CATEGORY
2- STATE WISE SALES
3- TOP TEN CITIES BY SALES
4- LOWEST PERFORMING CITIES
5- TOTAL SALES BY SEGMENT

TOOLS USED

1-EXCEL
2-PIVOT TABLES
3-PIVOT CHART
4-SLICERS

DASHBOARD FEATURE

1-INTERACTIVE SLICER REGION,CATEGORY,SEGMENT.
2-CLEAN KPI CARDS
3-USER-FRIENDLY LAYOUT

FILES INCLUDED
1-SUPERSTORE RAW DATA.XLSX
2-SALES ANALYSIS DASHBOARD.XLSX
3-PROJECT SCREENSHOT.PNG
